LINK'ed CSS

This is the main content. It has a cute watermark background. The HTML uses a LINKed stylesheet containing the following:

body { background-image: url(http://tool-man.org/mozilla-tests/test2/leaf-2-detroit.gif); }
#body { background-image: url(http://tool-man.org/mozilla-tests/test2/smiling-watermark-1.gif); }

The body background is a 200x200 pixel tiled leaf pattern. This "body" DIV's background is a 737x917 pixel watermark.

Reproducing the bug

Hit the reload button (literally hit the reload button, Ctrl-R reproduces the bug less often). Assuming you are on fast connection, the watermark "bleeds" below and to the right of this DIV's borders. Try copying this files to a local server to achieve a fast connection.

Try this same page, but with inline CSS (<-- you can keep clicking on this link and never reproduce the bug)

Now use the forward and back buttons. This sometimes reproduces the bug, but only when going back or forward to the LINK'ed CSS page.
